# 1. –ü—Ä–æ–µ–∫—Ç–∏–≤–Ω–∞—è –≥–µ–æ–º–µ—Ç—Ä–∏—è –∏ –æ–¥–Ω–æ—Ä–æ–¥–Ω—ã–µ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç—ã

**–¢–æ—á–∫–∏ –ø—Ä–æ–µ–∫—Ç–∏–≤–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏.** –†–∞—Å—Å–º–æ—Ç—Ä–∏–º –¥–≤—É—Ö–º–µ—Ä–Ω–æ–µ –ø—Ä–æ–µ–∫—Ç–∏–≤–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ (–∫–æ—Ç–æ—Ä–æ–µ –µ—â–µ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –ø—Ä–æ–µ–∫—Ç–∏–≤–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç—å—é). –í —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ –Ω–∞ –æ–±—ã—á–Ω–æ–π –ï–≤–∫–ª–∏–¥–æ–≤–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏ —Ç–æ—á–∫–∏ –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –ø–∞—Ä–æ–π –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç (*x*,*y*)*T*, –Ω–∞ –ø—Ä–æ–µ–∫—Ç–∏–≤–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏ —Ç–æ—á–∫–∏ –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è —Ç—Ä–µ—Ö–∫–æ–º–ø–æ–Ω–µ–Ω—Ç–Ω—ã–º –≤–µ–∫—Ç–æ—Ä–æ–º (*x*,*y*,*w*)*T*. –ü—Ä–∏ —ç—Ç–æ–º –¥–ª—è –ª—é–±–æ–≥–æ –Ω–µ–Ω—É–ª–µ–≤–æ–≥–æ —á–∏—Å–ª–∞ *a*, –≤–µ–∫—Ç–æ—Ä—ã (*x*,*y*,*w*)*T* –∏ (*ax*, *ay*, *aw*)*T* —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç –æ–¥–Ω–æ–π –∏ —Ç–æ–π –∂–µ —Ç–æ—á–∫–µ. –ê –Ω—É–ª–µ–≤–æ–π –≤–µ–∫—Ç–æ—Ä (0,0,0)*T* –Ω–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –Ω–∏–∫–∞–∫–æ–π —Ç–æ—á–∫–µ –∏ –≤—ã–∫–∏–¥—ã–≤–∞–µ—Ç—Å—è –∏–∑ —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–∏—è. –¢–∞–∫–æ–µ –æ–ø–∏—Å–∞–Ω–∏–µ —Ç–æ—á–µ–∫ –ø–ª–æ—Å–∫–æ—Å—Ç–∏ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –æ–¥–Ω–æ—Ä–æ–¥–Ω—ã–º–∏ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∞–º–∏ (homogeneous coordinates).

–¢–æ—á–∫–∞–º –ø—Ä–æ–µ–∫—Ç–∏–≤–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏ –º–æ–∂–Ω–æ —Å–æ–ø–æ—Å—Ç–∞–≤–∏—Ç—å —Ç–æ—á–∫–∏ –æ–±—ã—á–Ω–æ–π –ï–≤–∫–ª–∏–¥–æ–≤–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏. –ö–æ–æ—Ä–¥–∏–Ω–∞—Ç–Ω–æ–º—É –≤–µ–∫—Ç–æ—Ä—É (*x*,*y*,*w*)*T* –ø—Ä–∏ *w* 0 —Å–æ–ø–æ—Å—Ç–∞–≤–∏–º —Ç–æ—á–∫—É –ï–≤–∫–ª–∏–¥–æ–≤–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏ —Å –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∞–º–∏ (*x*/*w*, *y*/*w*)*T*. –ï—Å–ª–∏ –∂–µ *w* = 0, —Ç.–µ. –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–Ω—ã–π –≤–µ–∫—Ç–æ—Ä –∏–º–µ–µ—Ç –≤–∏–¥ (*x*, *y*, 0*T*), —Ç–æ –±—É–¥–µ–º –≥–æ–≤–æ—Ä–∏—Ç—å, —á—Ç–æ —ç—Ç–∞ —Ç–æ—á–∫–∞ –≤ –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ—Å—Ç–∏. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –ø—Ä–æ–µ–∫—Ç–∏–≤–Ω—É—é –ø–ª–æ—Å–∫–æ—Å—Ç—å –º–æ–∂–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å –∫–∞–∫ –ï–≤–∫–ª–∏–¥–æ–≤—É—é –ø–ª–æ—Å–∫–æ—Å—Ç—å, –¥–æ–ø–æ–ª–Ω–µ–Ω–Ω—É—é —Ç–æ—á–∫–∞–º–∏ –∏–∑ –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ—Å—Ç–∏.
